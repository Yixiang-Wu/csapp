1. 无符号数加法
    - 将二者转化为二进制数，按位相加，随后取模
2. 判断无符号数相加是否溢出(正溢出)
    - sum = x + y
    - 若sum < x 或者 sum < y，则说明溢出
3. 无符号数求反，或者阿贝尔逆元
    - $-x^u_w = 2^w - x$
4. 补码加法
    - 将二者转化为二进制数，按位相加，随后取模
    - 注意，若是负数，则：十进制数-->原码-->符号位不变，其他取反，末尾加1-->得到补码
5. 补码加法判断溢出
    - 若x>0, y>0, sum<=0则溢出
    - 若x<0, y<0, sum>=0则溢出
6. 补码的非
    - $-x = 2^w - x$
    - 所有二进制位取反，末尾加1
